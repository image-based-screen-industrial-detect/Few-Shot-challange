Submission Guidelines

Participants are required to submit a JSON file containing their predictions on the test set. Each prediction should follow this structure:

[
    {
        "image_id": "00001",
        "bbox": [x_min, y_min, width, height],
        "score": 0.95,
        "category_id": 1
    },
    ...
]
  • image_id: ID of the image (without file extension).
  • bbox: [x_min, y_min, width, height] format bounding box (pixel values).
  • score: Confidence score between 0 and 1.
  • category_id: Defect category number according to provided label map.

Submissions must adhere to this format for successful evaluation.
